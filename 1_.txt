**🔧 ビルド方法（2パターン）**

**パターン1: Visual Studio使わないでコマンドラインだけ**
```bash
# .NET SDKがない場合はインストール（無料）
# https://dotnet.microsoft.com/download

dotnet build OutlookMailMonitor.csproj
dotnet run --project OutlookMailMonitor.csproj
```

**パターン2: csc.exe直接使う（.NET Frameworkの場合）**
```bash
# Outlookの参照DLLを手動で追加する必要あり
csc /r:"C:\Windows\assembly\GAC_MSIL\Microsoft.Office.Interop.Outlook\15.0.0.0__71e9bce111e9429c\Microsoft.Office.Interop.Outlook.dll" OutlookMailMonitor.cs
```

**💡 オススメは.NET使う方！**
.NET 6以降なら、csprojファイルが自動でOutlookの参照を解決してくれるから超楽だよ！

**⚠️ 注意点**
- Outlookがインストールされてる必要あり
- Outlookが起動してる状態で実行してね
- 終了はEnterキーかCtrl+C
